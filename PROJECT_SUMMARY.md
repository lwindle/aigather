## 部署指南

### 快速部署
```bash
# 1. 上传文件到服务器
scp -r ./* root@your-server:/var/www/aigather/

# 2. 运行部署脚本
cd /var/www/aigather
chmod +x deploy.sh
./deploy.sh production
```

### 手动部署
```bash
# 1. 构建应用
go build -o aigather main.go

# 2. 配置Nginx
# 参考 README.md 中的详细步骤

# 3. 启动服务
sudo systemctl start aigather
sudo systemctl restart nginx
```
